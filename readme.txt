Automatically add rel="nofollow" and target="_blank" to all external links.

== Description ==

This plugin will add rel="nofollow" and target="_blank" to all external links automatically.

= Feature =
* Automatically Add rel=nofollow for all the external links of your blog/websit.
* Automatically Add target=_blank for all the external link of your blog/website.
* If you already added rel=dofollow to any post manually, this plugins will not add rel=nofollow for that post.
* If you already added target=_blank to any post manually, this plugins will not add target=_blank for that post.
* SEO Nofollow External will not add rel=nofollow and target=_blank any of the internal link of your blog/website.

Visit our website codeilike.com

== Installation ==

1. Download the latest zip file and extract the 'seo-nofollow-external' directory.
2. Upload it to your '/wp-content/plugins/' directory.
3. Activate 'SEO Nofollow External' on the 'Plugins' menu in WordPress.
4. That's it, you are all done :)
